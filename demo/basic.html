Here is a player for YouTube's own channel. This player is completely HTML + YouTube own IFRAME based embedding system, that will use HTML5 video on iPad, iPhone and other system that does not support Flash

You can customize the appearance quite easily, via CSS. Moreover, you can supply custom functions on options :

  • carousel: function($maindiv, videos, options) - creates the "carousel" of videos in the channel, the default function calls "thumbnail" for each video.
  • thumbnail: function($carouseldiv, video, options) - creates a thumbnail for a specific video, the default function creates an img pointing to the first available thumbnail image for the video, with a click handler that calls "player".
  • player: function($maindiv, video, options) - creates the player, the default function creates an IFRAME as specified by YouTube embedding system, using properties found in options.playopts as options.
  • loaded: function(videos, options) - called after the list of videos is called, the carousel is created and the player for the first video is displayed. The default implementation does nothing.
